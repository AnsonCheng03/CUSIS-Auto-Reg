Bookmarklet 操作說明

桌面瀏覽器（Google Chrome）

請將以下代碼複製並建立新書籤：

javascript:(function(){var script=document.createElement('script');script.src='https://ansoncheng03.github.io/CUSIS-Auto-Reg/script.js?time=' + new Date().getTime();document.body.appendChild(script);})();

進入 CUSIS 購物車頁面後，點擊書籤啟動自動登記功能。

iOS / iPadOS（Safari）

複製以下代碼並保存到書籤：

javascript:(function(){var script=document.createElement('script');script.src='https://ansoncheng03.github.io/CUSIS-Auto-Reg/script.js?time=' + new Date().getTime();document.body.appendChild(script);})();
  1. 使用 Safari 瀏覽器開啟此頁面，複製代碼後點擊分享按鈕 → 「加入書籤」。
  2. 編輯書籤並將網址替換為代碼，然後儲存變更。
  3. 進入 CUSIS 購物車頁面後，點擊書籤啟動功能。

Android（Google Chrome）

將以下代碼複製並添加到書籤：

javascript:(function(){var script=document.createElement('script');script.src='https://ansoncheng03.github.io/CUSIS-Auto-Reg/script.js?time=' + new Date().getTime();document.body.appendChild(script);})();
  1. 使用 Chrome 瀏覽器開啟此頁面，複製代碼後點擊右上角功能表（⋮）→ 加入書籤。
  2. 編輯書籤並將網址替換為代碼，然後儲存。
  3. 進入 CUSIS 購物車頁面後，點擊書籤啟動功能。
